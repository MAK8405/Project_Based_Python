# Number Guesser Game Project

## Description

This project is a game where users can guess a number between 1 and 100. The user will be given a hint if the number is too high or too low. The user will able be able to see the number of guesses they have made. The user will be able to reset the game at
any time.

## How to Run

1. Clone down this repo, and cd into the project
2. Run `pip install -r requirements.txt` in your command line
3. Run `python number_guesser.py` in your command line
